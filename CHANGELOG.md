# Changelog

All notable changes to the ARCHON Dev Engine will be documented in this file.

## [0.1.0] - 2026-02-25

### Added
- **Async-First Doctrine**: Rule to block blocking I/O in async contexts.
- **Structured Discipline**: Rule to mandate JSON schema adherence.
- **Deterministic Config**: Rule to enforce host environment configuration.
- **ARCHON Scorer**: CLI tool to check project compliance score.
- **ARCHON Guard**: Portable Git pre-commit hook template.
- **ARCHON Architect**: Specialized system designer agent.
- **Feedback Loop**: `archon feedback` command for bug reporting.
- **Psychological Guardrails**: Improved veto messaging for "Saved From Deployment" signals.
